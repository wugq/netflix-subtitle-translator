Netflix Subtitle Translator

Auto-translates English subtitles to Simplified Chinese using OpenAI GPT.

OpenAI API Key

How It Works

  1. Open a Netflix show — English subtitles will be auto-enabled.
  2. The extension intercepts the subtitle file and translates it via GPT.
  3. Chinese subtitles appear in the Netflix player.

Your API key is stored locally in your browser and never sent anywhere except OpenAI.
