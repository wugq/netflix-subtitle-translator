How It Works

  1. Open a Netflix show — select your source language in Netflix's player and your destination language in the extension popup.
  2. The extension intercepts subtitles and translates them via GPT, using the English track as input when available for better results.
  3. Translated subtitles are displayed in a custom overlay. If the destination language is natively available on Netflix, no AI translation is used.
  4. Translation is buffered progressively to minimise API usage. You can pause or resume translation at any time from the popup.

Your API key is stored locally in your browser and never sent anywhere except the configured AI provider.
